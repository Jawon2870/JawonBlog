1. 是外观很漂亮

缺点：

1. 项目很臃肿，难以维护。由于其集成功能过多，所包含依赖过多，导致项目打包速度较慢，不易维护。

3. Gitee 目前不在提供 Gitee pages 服务，导致所有依赖 Gitee pages 的博客都挂掉了，本文中的方案已不再可行。
